LoggingService

A generic service for logging. To use the LoggingService, simply add it as a constructor parameter. For example: 

constructor(private loggingService: LoggingService) {{ '{' }} {{ '}' }}

Make sure that the LoggingService is provided. Since all modules should make use of this generic service, app.module.ts is a good place to add the LoggingService.
