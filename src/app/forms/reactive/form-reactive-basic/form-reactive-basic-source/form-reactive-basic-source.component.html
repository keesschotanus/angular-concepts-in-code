Basic Form (Reactive)

This form builds upon the basic, template driven, form.

Note: To use a reactive Form, you have to import the ReactiveForms module. 


      import {{ '{' }} ReactiveFormsModule }} '}' }} from '@angular/forms';
      ...
      imports: [
        ...,
        ReactiveFormsModule,
        ...
      ],
      ...

Differences between this reactive form and its related, template driven form

The component is now quite different. It needs to create the form from Typescript code.

The HTML page is quite similar, but it too, requires some changes.

  1. You need to add [formGroup]="signupForm" to the form
  2. When submitting the form, you don't need to pass the form anymore
  3. You have to add the formControlName attribute to all inputs
  4. You can remove the ngModel attribute
