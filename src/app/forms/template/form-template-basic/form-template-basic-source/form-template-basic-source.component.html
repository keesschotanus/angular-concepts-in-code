Basic Form (Template Driven)

This component shows a form with different input elements. When you submit the form an NgForm object is written to the console.

The Form

The form definition has to look something like this: 

<form (ngSubmit)="onSubmitForm(theForm)" #theForm="ngForm"></form>

Normally you submit a form using something like method="POST", but this would send a request. To submit the form Angular style, use the (ngSubmit) event handler. Make sure to make the form available using a local reference. In this example I used "#theForm"". This Angular form is passed to the submit form handler that I named onSubmitForm.

Note: Instead of passing the form, you can also retrieve it in your component using: 

@ViewChild('theForm') theForm: NgForm

Note: To use a Form, you have to import the Forms module. 


      import {{ '{' }} FormsModule }} '}' }} from '@angular/forms';
      ...
      imports: [
        ...,
        FormsModule,
        ...
      ],
      ...

The Input Elements

In order for Angular to see the input item you created as a form element, the input item must have:

  • a name
  • the ngModel attribute
