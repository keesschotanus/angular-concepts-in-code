Form validation using Primefaces (Template Driven)

Compared to the Basic Primefaces (Template Driven) Form, this form adds validation. Additionally it shows how to use the message service. You can see it in action when you submit the form.

Differences between this form and the basic form

The differences are comparable to the differences between the basic form and the validation form, both non-Primefeaces that is. Here too, you will add the disabled attribute to the submit button and you will use *ngIf, to conditionally show error messages.

The nice thing about PrimeFaces is that you don't have to define your own styles, as they are all part of Primefaces.
