@ViewChild

@ViewChild can be used to inject a child component into a parent component. In this particular example the child component is injected like this: 

@ViewChild(ViewChildChildComponent) private childComponent: ViewChildChildComponent;

The parent now has access to the complete child component. You may want to implement AfterViewInit to make sure that you don't access the child before it is rendered.

Instead of passing a child component, you may also pass in a local reference to any element in your HTML template. So if your child's HTML contains: <input ... #inputReference>, you may use @ViewChild in your parent component like this: 

@ViewChild('inputReference') private childComponent: ElementRef;

Note that the type of the injected element is of type ElementRef which has a nativeElement property.
