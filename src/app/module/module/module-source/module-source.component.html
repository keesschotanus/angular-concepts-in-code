Angular Module

This is a demonstration of a simple module. Initially the module directory did not represent a module at all. I turned it into a module and in the process described all individual steps, to make this an actual Angular model.

Starting situation

Created a component with: ng g c module/module
Nested two components like this:
ng g c module/module/module-first
ng g c module/module/module-second
Both module-first and module-second, use the roman pipe.

  • @NgModule app.module.ts contains the Module components
  • Added the Module components to the menu in app.component.ts
  • Added the different routes to app-routing.module.ts

Converting the module directory to an actual module

  1. Created a feature.module.ts file and turned this into a module
    • Added an import of the CommonModule
  2. Removed the Module components from the app.module.ts file and the corresponding imports
  3. Created a new feature-routing.mdoule.ts file
    • Moved the module related routes from app-routing.module.ts in this file and removed the imports
  4. Imported the FeatureRoutingModule in feature.module.ts
  5. Imported the FeatureRoutingModule in app.module.ts

When you now run the application, you will see the next error in the debug window of your browser: Type RomanPipe is part of the declarations of 2 modules:
FeatureModule and AppModule! Please consider moving RomanPipe to a higher module that imports FeatureModule and AppModule.
You can also create a new NgModule that exports and includes RomanPipe then import that NgModule in FeatureModule and AppModule.

I took the latter approach and created a new SchotanusSharedModule that exports the RomanPipe. Both app.module.ts and feature.module.ts import the RomanPipe.
