Local Reference

A local reference can be used to allow a parent component, to access the child component. This example contains two local references.

The first is a reference on the input item. When you click the button, a reference to the input item is passed as a parameter to the onClick(...) method.

The second reference is added to the child component like this: 

<app-local-reference-child #childReference></app-local-reference-child>

This allows you to use code like this in your parent template: 

{{ '{' }}{{ '{' }} childReference.getValueFromChild() {{ '}' }}{{ '}' }}
