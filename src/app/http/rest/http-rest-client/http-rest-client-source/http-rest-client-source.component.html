HTTP Client

Angular's HTTP client allows you to execute all the HTTP methods, like DELETE, GET, POST, PUT and PATCH. Pay special attention to the periodic-table.service.ts file and http-rest-client.component.ts. The former actually uses the HTTP client while the latter subscribes to the response.

To use http I imported the HttpModule in app.module.ts like this:
import {{ '{' }} HttpModule {{ '}' }} from '@angular/http';
The service uses:
import {{ '{' }} Http, Response {{ '}' }} from '@angular/http';

Note: This example uses NodeJS' express server.
