HTTP Client

Angular's HTTP client allows you to execute all the HTTP methods, like DELETE, GET, POST, PUT and PATCH. Pay special attention to the firestore-periodic-table.service.ts file and http-rest-client.component.ts. The former actually uses the HTTP client while the latter subscribes to the response.

To use http I imported the HttpModule in app.module.ts like this:
import {{ '{' }} HttpModule {{ '}' }} from '@angular/http';
The service uses:
import {{ '{' }} Http, Response {{ '}' }} from '@angular/http';

Note: This example uses Google's Cloud Firestore database. The nice thing about this is that I can demonstrate my code everywhere since the database is always up and running. The down side is that this database uses documents instead of javascript objects. Other interfaces solve this problem but I wanted to use the REST interface in combination with the HTTP client.
