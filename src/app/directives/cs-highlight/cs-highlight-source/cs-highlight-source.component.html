cs-highlight

The cs-highlight directive can be used to highlight elements. It is for demonstration purposes only since you should be able to highlight using CSS.

Notice that @HostListener is used to bind to listen for events originating from the host. @HostBinding is used to bind to a DOM property.

Usage: <e cs-highlight>content</e> to use the default highlight color.
<e [cs-highlight]="color-expression">content></e> to use your own color.
