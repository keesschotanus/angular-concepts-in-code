cs-code

The cs-code directive can be used to format source code. It is an alternative to using <pre><code>...</code></pre>

With pre and code you have to align you source code against the left margin, to avoid getting superfluous spaces in your formatted code. Not so with <cs-code> which allows you to properly indent your source. The cs-code directive will take care of removing these spaces automatically.

Unfortunately it doesn't work at all since it can't be used with un-escaped accolades for example. Nevertheless it is a good example of what directives are capable of.

Usage:
<cs-code>
  Your source code nicely indented here.
</cs-code>
