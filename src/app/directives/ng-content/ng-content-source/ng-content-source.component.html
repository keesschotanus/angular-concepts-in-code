ng-content

Normally when you use a component like this: 


        <app-panel>
          <p>Content</p>
        </app-panel>

The complete content is removed by angular. ng-content allows you to insert content from the parent component into the child component. I created a simple panel component to demonstrate this. Check out the source of ng-content-panel.component.html. You should see this code for the header: 

<ng-content select="[facet=header]"></ng-content>

For the body you should simply see:

<ng-content></ng-content>

Note that there is no selector for the body of the panel. To use the panel, use code similar to this: 


        <app-ng-content-panel>
          <h1 facet="header">This is the header</h1>
          <p>This is the body</p>
        </app-ng-content-panel>

Notice that the header requires facet="header" where the body is included by default.

Selectors

Instead of using an attribute selector you could use an HTML element selector, but angular would not recognize it. The result would be: "Unhandled Promise rejection: Template parse errors: 'element-name' is not a known element. You can change te schema metadata property in your module if you still want to select an element.
